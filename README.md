## 前言

交换时间只剩一个多月了，十二月末还有圣诞集市和土耳其的旅行，其他旅程都已结束，我也开始准备期末考试了，以上的内容就是我交换旅行的主体部分啦。大多数像是记流水账，也有部分是有感而发。Anyway，往后应该时常会回忆起这段美妙的时光，到时候看到这些文字照片时，这一百零一夜仍如初见。
